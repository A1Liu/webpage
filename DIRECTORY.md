# Directory Guide
The directory structure is:

```
.
├── DIRECTORY.md
├── README.md
├── ReactTutorial.md
├── gulpfile.js
├── package-lock.json
├── package.json
├── public
│   ├── favicon.ico
│   ├── index.html
│   └── manifest.json
└── src
    ├── App.js
    ├── App.test.js
    ├── assets
    │   ├── css
    │   │   └── default.min.css
    │   └── scss
    │       ├── _config.scss
    │       └── default.scss
    ├── components
    │   ├── footerComponent
    │   │   └── footer.js
    │   ├── headerComponent
    │   │   └── header.js
    │   └── pages
    │       ├── homepage.js
    │       ├── bored.js
	│	  	├── projects.js
    │       └── resources.js
    ├── index.js
    └── registerServiceWorker.js
```
